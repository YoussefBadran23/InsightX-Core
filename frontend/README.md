# Frontend — Next.js 14 (App Router)

Built with **Next.js 14**, **Tailwind CSS**, **Recharts**, and **Zustand**.

## Structure

```
frontend/
├── app/               # App Router pages & layouts
├── components/        # Reusable UI components
├── lib/               # API clients, utils, Zustand stores
├── public/            # Static assets
├── styles/            # Global CSS / Tailwind config
├── package.json
├── tailwind.config.ts
├── next.config.ts
└── .env.example
```

## Local Dev

```bash
npm install
npm run dev   # → http://localhost:3000
```
